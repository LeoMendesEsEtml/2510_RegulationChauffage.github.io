Description détaillée

Module de test des entrées/sorties 24V pour contacts secs.

Ce module fournit des fonctions pour tester et valider les contacts secs via les interfaces I/O 24V. Permet de détecter l'état des contacts (ouvert/fermé) sur deux canaux indépendants avec gestion des GPIO et temporisations appropriées.
