Description détaillée

Mapping des résistances TMUX par type de sonde de température.

Ce module fournit les tables de correspondance entre les canaux 0-31 du multiplexeur TMUX et les valeurs de résistance réelles pour différents types de sondes de température. Chaque type de sonde dispose d'un mapping spécifique avec 32 valeurs calibrées.
