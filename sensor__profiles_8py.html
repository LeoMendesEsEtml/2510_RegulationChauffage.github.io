Description détaillée

Profils capteurs extraits de 'Calcul ADC.xlsx' (Feuil1).

Ce module contient les profils de configuration pour différents types de capteurs de température avec leurs paramètres ADC optimisés (Rref, IDAC, PGA) et leurs tables de conversion température/résistance. Les données proviennent des calculs d'optimisation du fichier Excel de référence.
