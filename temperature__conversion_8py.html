Description détaillée

Module de conversion résistance/température via interpolation linéaire.

Ce module fournit les fonctions de conversion bidirectionnelle entre résistance mesurée et température pour différents types de capteurs. Utilise l'interpolation linéaire sur les tables de calibration et supporte la conversion inverse pour la simulation de résistance.
